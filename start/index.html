Before You Book / Come Prepared

Session time is strict

Session time is strict. Please have lyrics, beats, files, references, instruments, and project notes ready before the session.

  • Late arrival does not extend the session.
  • Writing during the session counts as session time.
  • Beat searching or file downloading counts as session time.
  • Extra time is $25/hr when available.
  • Heavy edits or complex revisions may cost extra.
  • Larger projects are reviewed before final scheduling.

FAQ

Common questions

Is this a commercial studio?

Leaning Studios is currently a private, appointment-based recording setup for independent artists, bands, and creators.

Do packages include mix/master?

Customer packages include basic mix/master finishing. More detailed editing, advanced vocal production, or heavy revisions may require an upgrade.

What if I do not know which package I need?

Start the form and choose “Not sure yet.” Leaning Studios will help route the project based on song count and what you need.

Do bands cost more?

Yes. Band packages start higher because setup, coordination, file handling, and mix complexity are usually greater.
